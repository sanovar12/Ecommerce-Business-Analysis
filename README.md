# Ecommerce-Business-Analysis
End-to-end business analysis project covering sales performance, customer insights, SQL analysis and Power BI dashboarding.

## Project Overview

This project analyses e-commerce sales data to identify opportunities for revenue growth and profitability improvement.

## Business Problem

Management has observed fluctuating sales performance and inconsistent profitability across product categories and regions.

The objective is to identify:

- Top performing products
- High value customers
- Regional performance trends
- Profitability opportunities

- SQL
- Power BI
- Excel
- GitHub

## Deliverables

- KPI Analysis
- Customer Analysis
- Product Analysis
- Regional Analysis

- Business Recommendations
